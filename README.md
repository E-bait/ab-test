# A/B-тест: влияние рекламы на конверсию

## Анализ в Jupyter Notebook

https://github.com/E-bait/ab-test/blob/main/AB_test.ipynb

## О проекте

Реальные данные маркетингового A/B-теста: ~588 тысяч пользователей.
Проведён A/B-тест для оценки эффективности рекламной кампании.  
В тесте участвовало 588 101 пользователей, разделённых на две группы:

- **ad** — видели рекламу продукта
- **psa** — видели рекламу, но не самого продукта

**Цель** — проверить, повышает ли реклама конверсию в покупку.

## Результаты

Конверсия в группе с рекламой оказалась выше — **2.55%** против **1.79%** в контрольной группе.  
Разница составляет **+0.76 п.п**.

Статистический тест (t-test) показал p-value < 0.001 — разница значима, результат не случаен.

**Вывод:** рекламная кампания эффективна и приводит к росту покупок.
